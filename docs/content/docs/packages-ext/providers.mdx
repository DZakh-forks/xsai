## Cloud

<PkgSizeBadge pkg="@xsai-ext/providers-cloud" />

```package-install
npm i @xsai-ext/providers-cloud
```

### Examples

#### Keys

Checkout all supported providers [here](https://github.com/moeru-ai/xsai/tree/main/packages-ext/providers-cloud/src/providers).

```ts twoslash
import * as providers from '@xsai-ext/providers-cloud'

type Keys = keyof typeof providers
```

#### OpenAI

```ts twoslash
import { createOpenAI } from '@xsai-ext/providers-cloud' // [!code ++]
import { generateText } from '@xsai/generate-text'
import { env } from 'node:process'

const openai = createOpenAI(env.OPENAI_API_KEY!) // [!code ++]

const { text } = await generateText({
  ...openai.chat('gpt-4o'), // [!code ++]
  apiKey: env.OPENAI_API_KEY!, // [!code --]
  baseURL: 'https://api.openai.com/v1/', // [!code --]
  messages: [{
    content: 'Why is the sky blue?',
    role: 'user'
  }],
  model: 'gpt-4o', // [!code --]
})
```
